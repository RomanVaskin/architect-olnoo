## Назначение документа

Этот документ фиксирует контракт API: доступные эндпоинты, форматы данных, коды ошибок и требования к аутентификации/авторизации.

## Аутентификация

Каждый маршрут ниже требует подтверждённую Supabase cookie-сессию (см. `specs/authentication.md`) через `requireAuthenticatedUser` (`src/lib/auth.ts`); `local-development` bypass намеренно отклоняется маршрутами `/api/projects/*`, если запущен реальный Supabase-backend. Ответ на ошибку всегда имеет форму `{ "error": { "code": string, "message": string } }` с безопасным сообщением — сырые ошибки Postgres/Storage никогда не попадают в ответ.

## Проекты (`src/app/api/projects/`)

| Маршрут | Метод | Назначение | Коды ошибок |
|---|---|---|---|
| `/api/projects` | GET | Список облачных проектов текущего пользователя | 401, 503 |
| `/api/projects/:projectId` | GET | Детали проекта (файлы, ракурсы, концепции, версии, отзывы, события, подписанные URL изображений) | 401, 404, 503 |
| `/api/projects/:projectId/selected-concept` | POST | Сохраняет выбранную основную концепцию (`{ conceptId: string \| null }`) | 400, 401, 404, 503 |
| `/api/projects/:projectId/feedback` | POST | Добавляет отзыв к концепции (`{ conceptId, comment }`, до 5000 символов) | 400, 401, 404, 503 |
| `/api/projects/:projectId/files/:fileId/url` | GET | Перевыпускает подписанный URL файла (восстановление после истечения) | 401, 404, 503 |
| `/api/projects/import/prepare` | POST | Готовит идемпотентный план переноса локального проекта (см. `specs/server-project-sync.md`) | 400, 401, 413, 503 |
| `/api/projects/import/complete` | POST | Подтверждает загруженные файлы и сохраняет связанные метаданные | 400, 401, 503 |

`404` намеренно не отличает «проект не существует» от «проект существует, но принадлежит другому пользователю» — RLS возвращает пустой результат в обоих случаях, и различать их в ответе означало бы утечку факта существования чужого проекта.

## Концепции (`src/app/api/concepts/`)

| Маршрут | Метод | Назначение |
|---|---|---|
| `/api/concepts/generate` | POST | Платная генерация концепций (Gemini) для локального проекта |
| `/api/concepts/correct` | POST | Платное исправление концепции по замечаниям Quality Gate (только локальные проекты) |
