# Supabase

Миграции в этой папке являются версионируемой схемой будущего backend Architect OLNOO.

`migrations/202607200001_backend_foundation.sql` подготовлена для ревью, но **не применена** к удалённому Supabase-проекту. Перед `supabase db push` необходимо отдельно подтвердить целевой проект, проверить SQL/RLS и создать резервную копию существующей схемы.
