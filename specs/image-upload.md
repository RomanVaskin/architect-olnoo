## Назначение документа

Описывает требования к функции загрузки изображений: поддерживаемые форматы, ограничения по размеру, валидацию и последующую обработку.

## MVP

- поддерживаются JPG, PNG, WebP и PDF;
- максимальный размер одного файла — 20 МБ;
- один проект принимает до 12 файлов за один шаг;
- дубликаты по имени и размеру не добавляются повторно;
- неподдерживаемый формат и превышение лимита показываются пользователю до запуска проекта;
- для продолжения требуется хотя бы один исходный файл.

До подтверждения проекта файлы существуют только в состоянии мастера. При создании проекта байты растровых фотографий сохраняются в IndexedDB браузера (см. [multi-view-source-pipeline.md](multi-view-source-pipeline.md)) и переживают обновление страницы — это по-прежнему временное frontend-хранилище, а не Object Storage будущего бэкенда. PDF-файлы сохраняются только как метаданные, без байт.
